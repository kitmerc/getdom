# getdom
a little bash script for getting only the domain from any link


1
*********

just copy the hm file to the utilities section and give the rights

Example:

```
$ cd getdom

$ chmod +x hm

$ cp hm /usr/bin/hm
```
********

use the link as a parameter

Example:

```
$ hm https://github.com/kitmerc/getdom
github.com
```
